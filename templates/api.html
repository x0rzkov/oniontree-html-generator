Services

Service's data are available as JSON. You can access them by replacing .html with .json in URL.

Example

{{- if eq (getTarget) "clearnet" }} 
$ curl https://oniontree.org/services/nytimes.json
{{ else }} 
$ curl http://onions53ehmf4q75.onion/services/nytimes.json
{{ end }}

PGP keys

You can download PGP keys, if you know their ID.

Example

{{- if eq (getTarget) "clearnet" }} 
$ curl https://oniontree.org/keys/E9971784146CCD94.txt
{{ else }} 
$ curl http://onions53ehmf4q75.onion/keys/E9971784146CCD94.txt
{{ end }}
