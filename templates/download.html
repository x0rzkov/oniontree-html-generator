Download

All data published on this site are open source. You can download a copy from a repository hosted on GitHub.

$ git clone https://github.com/onionltd/oniontree

Bookmarks

If you visit this site often, you may be interested in importing OnionTree Bookmarks to save time. Once imported, typing a name of service or a tag in address bar will suggest the right page.

Importing in Tor Browser

Click OnionTree Bookmarks to save the file on your disk.

Press CTRL + SHIFT + H to open your library.

Click 'Import and Backup' to expand available options and choose 'Import Bookmarks from HTML...'.

Navigate to location where you previously saved bookmarks.html and select the file in order to import it.

That's it. You should now have a new directory named OnionTree in 'Bookmarks Menu'.

If you are a Tails user, don't forget to configure your persistent volume in order to preserve bookmarks between reboots.
